# CI/CD Pipeline using Jenkins & AWS

## Overview
This project demonstrates a CI/CD pipeline using Jenkins for building, testing, and deploying an application.

## Tools Used
- Jenkins
- AWS EC2
- Maven
- GitHub
- Tomcat

## Pipeline Stages
- Build
- Test
- Package
- Deploy

## Features
- Automated build and deployment
- Integrated with GitHub
- Multi-stage pipeline
